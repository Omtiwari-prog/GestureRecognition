# GestureRecognition
A real-time sign language recognition system that detects hand gestures and converts them into text using Computer Vision, OpenCV, Media Pipe, TensorFlow, and Machine Learning techniques.
# Sign Language Recognition and Text Conversion

## Overview
This project converts sign language gestures into text using Computer Vision and Machine Learning.

## Features
- Real-time hand tracking
- Gesture recognition
- Text generation
- OpenCV integration
- MediaPipe hand detection

## Technologies Used
- Python
- OpenCV
- MediaPipe
- TensorFlow
- NumPy

## How to Run

pip install -r requirements.txt

python datacollection.py
